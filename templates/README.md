# Document Templates

This folder contains reusable `.docx` templates for document generation.

## Available templates

- `InvestmentContract.docx`

## InvestmentContract placeholders

Request-backed placeholders:

- `{{ContractDate}}`
- `{{LenderFullName}}`
- `{{FirstName}}`
- `{{LastName}}`
- `{{CompanyName}}`
- `{{InvestmentAmount}}`
- `{{EquityPercentage}}`

Configuration-backed placeholders:

- `{{BorrowerCompanyName}}`
- `{{BorrowerCompanyAddress}}`
- `{{BorrowerRegisterNumber}}`
