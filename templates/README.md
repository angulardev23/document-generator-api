# Document Templates

This folder contains reusable `.docx` templates for document generation.

## Available templates

- `InvestmentContract.docx`

- `{{ContractDate}}`
- `{{FirstName}}`
- `{{LastName}}`
- `{{CompanyName}}`
- `{{InvestmentAmount}}`
- `{{EquityPercentage}}`
